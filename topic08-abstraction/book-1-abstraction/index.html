Network-V5

In this step, you will create version 5 of the Network system (our menu driven messaging system). In this version, you will use abstraction in the inheritance hierarchy.

Abstraction in Network-V5

In IntelliJ, create a new project called Network-V5.

Copy the following classes into this new project:

- [PhotoPost](archives/v4/PhotoPost.java)
- [MessagePost](archives/v4/MessagePost.java)
- [Post](archives/v4/Post.java)
- [NewsFeed](archives/v4/NewsFeed.java)
- [Driver](archives/v4/Driver.java)

Make the Post class abstract and add an abstract method called displayExtract() to it. The return type of this method should be a String.

In Post's subclasses, you will need to provide the concrete implementation for this method; this will eliminate the compile errors that we have now generated!

Message Post code:

    public String displayExtract()
    {
        return "Message extract: "+ message.substring(0,10) + "....";
    }

PhotoPost code:

    public String displayExtract()
    {
        return "Photo caption: " + caption.substring(0,10) + "....";
    }

Network-V5

In this step, you will use the new method we just added.

NewsFeed

In your NewsFeed class, add the following method:

    public String showExtract()
    {
        String str = "";
        // display all posts
        for(Post post : posts) {
            str += post.displayExtract() + "\n\n";
        }
        return str;
    }

Driver

In your Driver class, add a new menu option, number 4:


    private int mainMenu() {
        System.out.println("1) Add a Text Post");
        System.out.println("2) Add a Photo Post");
        System.out.println("3) List all Posts");
        System.out.println("4) List an extract of all Posts");
        System.out.println("0) Exit");
        System.out.print("==>>");
        int option = sc.nextInt();
        return option;
    }

Then, in the run() method, add a new option 4 to the switch statement:

                switch (option) {
                    case 1:
                        addMessagePost();
                        break;
                    case 2:
                        addPhotoPost();
                        break;
                    case 3:
                        showPosts();
                        break;
                    case 4:
                        showExtractPosts();
                        break;
                }

Finally, include this new method:

   public void showExtractPosts() {
        System.out.println(newsFeed.showExtract());
    }

Save and Test your Code

Save your code.

Run your app. Add a series of message and photo posts. Then call option 3 to display all the posts. Now call option 4 to display an extract of your posts. You should now see the difference between the two menu options.
